# POS.AI Activation Portal

## Description
Une application web ultra-légère conçue pour l'activation des terminaux de point de vente (POS) de la suite POS.AI. L'objectif est de fournir une interface simple, rapide et infaillible pour que les techniciens ou les utilisateurs finaux puissent enregistrer leur matériel.

## Démarrage Rapide

Cette application utilise React et Vite. Elle est conçue pour être déployée comme une SPA (Single Page Application).

### Prérequis
- Node.js (v16 ou supérieur recommandé)
- npm ou yarn

### Installation et Lancement
L'application est prête à l'emploi via l'architecture existante. Aucune configuration complexe n'est requise pour le développement local.

## Structure du Projet
- `/components` : Composants React (UI générique et formulaires métier).
- `/services` : Logique d'appel API (actuellement simulée).
- `/docs` : Documentation technique et feuille de route.

## Stack Technique
- **Framework** : React 18+
- **Build Tool** : Vite
- **Styling** : Tailwind CSS
- **Icônes** : Lucide React
- **Langage** : TypeScript
